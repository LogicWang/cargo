# cargo
==================
pure c code  without any package build tool, just for practice
each package in src is independent.
can compile each single package and run it 
if you want to run any single program
step1. checkout code
```bash
git clone https://github.com/LogicWang/cargo.git 
```
step2. switch to corresponding dir
for example
```bash
cd ./src/kr/
```
step3.
```bash
gcc -o myname xxx.c  
```

step4. run compiled program
